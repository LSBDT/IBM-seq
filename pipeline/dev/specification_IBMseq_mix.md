# mixデータを解析できるようにする
これまでのpipeline実装と全く同じプロセスを行い、結果のPDFなども出力できるようにする。
コードの修正がシンプルにすまない場合はmix用の専用コードにわけていい。その場合は 01_mix_ などのmixネーミングを入れること
- 現在のデータは、抗体単体の情報の解析
- mixは一つのランに複数の抗体を加えて解析したもの
- V5P2_24aB_mix_2_3000.link.gzがそのテストデータ
- 元のコード、/old_script/IBMseq_process_data10.1.r にはmix modeがあるはずそれを一応参考にして。
- 基本的にデータの違いとして、以下だけが違う: df_membership$sum <- rowSums(df_membership[, 2:13], na.rm = TRUE) #抗体の数で変わる。5種類x5種類なら2:13 
- 今あるデータは抗体５種類だけど、４種類などのデータも増える可能性もあるので、その点を考慮したプログラム
- smモードもできるように
- 最後にテストデータで検証PDFを得てください。コードを修正する場合は、singleモードも動くことをチェック。
